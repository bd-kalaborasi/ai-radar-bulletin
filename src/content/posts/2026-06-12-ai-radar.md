### Cursor Bugbot upgraded: 3× faster, 22% cheaper, 10% more bugs detected

**Source:** https://www.cursor.com/changelog · Cursor · 2026-06-10
**Verification:** T2 verified · changelog · dev-tools

Cursor shipped Bugbot performance improvements on June 10, powered by the Composer 2.5 model. Review time dropped from approximately five minutes to approximately 90 seconds. Cost per review decreased 22%. Bug detection increased from 0.56 to 0.62 bugs per review. A new `/review` slash command lets developers run Bugbot before pushing code, with an option to review only changes since the last review rather than the full diff. The changelog was confirmed directly from the Cursor changelog page.

**Why it matters for automation/productivity:** Automated code review now adds under two minutes per PR cycle for teams using Cursor. The `/review` command integrates directly into pre-push workflow steps and can be chained into Claude Code or CI agent loops as a lightweight quality gate before deployment.

**Key claims:**
- Review time: ~5 min → ~90 sec (3× faster) → cursor.com/changelog (T2)
- Cost: 22% lower per review → cursor.com/changelog (T2)
- Bug detection: 0.56 → 0.62 bugs/review → cursor.com/changelog (T2)

### GitHub MCP Server 1.3.0 ships commit and blame tools for agents

**Source:** https://github.com/github/github-mcp-server/releases · GitHub · 2026-06-11
**Verification:** T2 verified · changelog · mcp-ecosystem / dev-tools

GitHub released version 1.3.0 of its official MCP Server on June 11. New additions: a `get_commits` method on the pull request write tool, letting agents retrieve commit history for a PR without leaving the MCP protocol surface; `get_file_blame` in insiders mode, linking source lines to their authoring commits. Rate limit error messaging was rewritten to surface clear, actionable errors to AI agents approaching the GitHub API limit — previous errors were opaque. Cursor pagination was added for Dependabot alerts, enabling agents to scroll through large security alert sets.

**Why it matters for automation/productivity:** `get_commits` removes a common break point in code-review agent pipelines, where agents previously had to exit MCP and invoke shell commands to fetch git history. Agents building automated PR summaries, changelogs, or risk assessments can now retrieve commit metadata within a single MCP session.

**Key claims:**
- `get_commits` added to PR write tool → github.com/github/github-mcp-server/releases (T2)
- `get_file_blame` added in insiders mode → same source
- Rate limit messaging improved for AI agents → same source
- Cursor pagination for Dependabot alerts → same source

### Anthropic launches Claude Corps national fellowship

**Source:** https://www.anthropic.com/news/claude-corps · Anthropic · 2026-06-11
**Verification:** T2 verified · announcement · ai-for-business

Anthropic announced Claude Corps on June 11, a national fellowship placing 1,000 AI-trained professionals in U.S. nonprofits over 12-month terms. Fellows earn $85,000 per year plus benefits and receive intensive onboarding plus five hours of ongoing weekly education. The program commits $150 million. The first cohort of 100 fellows starts in October 2026; applications close July 17. Additional cohorts begin January 2027 and August 2027. More than 400 nonprofits are confirmed as host organizations across food banks, veteran support groups, educational nonprofits, and marine conservation organizations. Applicants must be at least 18 years old with fewer than two years of full-time work experience and be authorized to work in the United States; no specific educational background is required.

**Why it matters for automation/productivity:** Informational only — no immediate workflow leverage. For nonprofit and civic-tech organizations, the program creates a pipeline of Claude-trained practitioners starting October 2026, lowering the cost of AI adoption in resource-constrained settings.

**Key claims:**
- $150M program commitment → Anthropic primary (T2)
- $85K/year + benefits per fellow → Anthropic primary (T2)
- First cohort: 100 fellows, October 2026 start → Anthropic primary (T2)
- Applications close: July 17, 2026 → Anthropic primary (T2)
- 400+ confirmed host nonprofits → Anthropic primary (T2)
